# Speech Recognition System

## CODTECH Internship Task 2

This project implements a basic Speech-to-Text system using Python.

The system captures audio from the microphone and converts speech into text using the SpeechRecognition library.

## Technologies Used

- Python
- SpeechRecognition
- PyAudio

## Files in Project

speech_to_text.py – main program  
output.txt – generated text output  
requirements.txt – required libraries  

## How to Run

1. Install dependencies

pip install SpeechRecognition pyaudio

2. Run the program

python speech_to_text.py

3. Speak into the microphone

The system will convert speech to text.
